# Trending Posts Archive

This directory contains automatically generated trending posts.
Each file follows the format: `YYYY-MM-DD-topic-slug.md`

Files are generated by the GitHub Actions workflow `.github/workflows/media-kit-trending.yml`
when trending academic topics are detected.

---

## Format

Each trending post file contains:
- YAML frontmatter (date, topic, category, platforms)
- Platform-specific post content (LinkedIn, Twitter, Instagram, Facebook, WhatsApp)

## Categories
- `scholar` — Birth/death anniversaries of notable scholars
- `event` — Academic events and international days
- `festival` — Cultural and educational festivals
- `breaking` — Major academic news and discoveries
