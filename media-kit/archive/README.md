# Media Kit Archive

This directory stores archived quarterly content from previous quarters.

Each archived quarter is moved here by the GitHub Actions workflow
`.github/workflows/media-kit-quarterly.yml` at the start of a new quarter.

## Structure

```
archive/
├── quarter-2026-Q3/
│   ├── social-posts/
│   ├── images/
│   ├── press-release.md
│   └── newsletter-draft.md
├── quarter-2026-Q4/
│   └── ...
└── ...
```
